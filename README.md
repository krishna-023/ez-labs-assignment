This is an enhanced, production-ready React + Vite + Tailwind project prepared for the EZ Labs frontend assignment.
It includes a responsive homepage, polished UI, and an integrated contact/apply form that POSTs to the provided API.

## Features
- Responsive across 480p, 720p, 1080p, iPad (2732x2048) and MacBook sizes
- Front-end validation: email required and validated
- Axios POST to: https://vernanbackend.ezlab.in/api/contact-us/
- Tailwind CSS styling with glassy cards and gradients
- Ready for Vercel deployment

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

A sample Postman collection `postman_collection.json` is included demonstrating the contact POST request.
